# Input Data Templates

本目录提供可直接填写的 CSV 模板，确保数据格式符合 jamovi-analysis 的预处理和分析要求。

## 通用规则

1. **宽格式（Wide Format）**：每行代表一个被试，每列代表一个变量。
2. **表头**：第一行必须是列名。支持中文表头（会自动别名化）。
3. **缺失值**：可留空，或使用 `NA`、`N/A`、`null`、`.`。
4. **量表题**：必须是整数（如 1–5 或 1–7），不允许小数。
5. **分组变量**：用于 t 检验时必须恰好 2 个水平；用于 ANOVA 时需 2 个及以上水平。

## 模板说明

### prepost_scale_study.csv
**适用场景**：教育实验中的前测/后测量表研究。

| 列名 | 说明 | 约束 |
|------|------|------|
| `user_id` | 被试唯一标识 | 不可重复 |
| `group` | 实验组/对照组 | 可选，2 个水平 |
| `pre_q01` ~ `pre_qNN` | 前测题项 | 整数，1~max_scale |
| `post_q01` ~ `post_qNN` | 后测题项 | 整数，与 pre 对应 |

**JobFile 配置要点**：
```json
{
  "preset": {
    "name": "prepost_scale_study",
    "id_column": "user_id",
    "group_column": "group",
    "max_scale": 5,
    "reverse_items": ["q03"],
    "subscales": {
      "creativity": ["q01", "q02", "q03"]
    }
  }
}
```

### cross_sectional_survey.csv
**适用场景**：横断面问卷调查。

| 列名 | 说明 | 约束 |
|------|------|------|
| `user_id` | 被试唯一标识 | 不可重复 |
| `gender` | 性别 | `1`/`2` 或 `男`/`女` |
| `age` | 年龄 | 整数 |
| `q01` ~ `qNN` | 量表题项 | 整数 |

### ttest_two_group.csv
**适用场景**：独立样本 t 检验。

| 列名 | 说明 | 约束 |
|------|------|------|
| `group` | 分组变量 | 恰好 2 个水平 |
| `score` | 连续因变量 | 可为小数 |

### reliability_scale.csv
**适用场景**：信度分析（Cronbach's α）。

| 列名 | 说明 | 约束 |
|------|------|------|
| `q01` ~ `qNN` | 量表题项 | 整数，使用同一量尺 |

### regression_study.csv
**适用场景**：线性回归或二元 logistic 回归。

| 列名 | 说明 | 约束 |
|------|------|------|
| `y` / `dep` | 因变量 | 连续（线性）或二分类（logistic） |
| `x1`, `x2` | 连续自变量 | 可为小数 |
| `gender` | 分类自变量 | 2 个及以上水平 |

## 常见错误

| 错误 | 现象 | 修复 |
|------|------|------|
| 量表列含小数 | 预处理报错 | 检查数据录入，确保为整数 |
| t 检验分组变量有 3 个水平 | 测量类型校验失败 | 改用 ANOVA (`anovaOneW`) |
| 反向题命名不一致 | 无法识别反向计分 | 确保 JobFile 中的 `reverse_items` 与列名匹配 |
| 缺失值写成 `999` | 被当作真实值分析 | 替换为空白或 `NA` |
